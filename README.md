# simplagion
=======================
Module with some functions for the generation of random simplicial complexes (https://arxiv.org/abs/1810.07031)

Dependencies
------------
* Numpy 1.15.3
* NetworkX 2.1


References
----------
I. Iacopini et al. (2018) Simplicial Models of Social Contagion. arXiv preprint arXiv:1810.07031.
